THE CONTENTS OF INTERPOLANT.TAR.GZ:

intepolant.py - the main file
interpolant-extended - the main program extended to deal with any number of variables
interpolant_check.py - a collection of functions used to run the experiments
interplant_plot.py - a collection of functions for plotting the final data from files with the suffix '_final.py

results50000.py - the output of interpolant(X,Y) in the main file, for 50000 formulas with c,d = 20
results50000c_final.py - the output of the main testing functions in interpolant_check.py for the case of results50000.py
results100000.py - the output of interpolant(X,Y) in the main file, for 50000 formulas with c,d \leq 10
results100000c_final.py - the output of the main testing functions in interpolant_check.py for the case of results100000.py [unadulterated]
results100000b_final_changed.py - the output of the main testing functions in interpolant_check.py for the case of results100000.py [two cases removed]

results1000.py - the output of interpolant(X,Y) in the extended file for 1000 formulas
results 1000_final.py - the output of the testing functions in interpolant_check.py for the case of results1000.py 

truthtable.py - a collection of functions used in validity checking
